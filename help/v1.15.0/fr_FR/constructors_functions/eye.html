eye
Crée une matrice identité.
📝Syntaxe
R = eye
R = eye(n)
R = eye(n, m)
R = eye(n, m, ..., z)
R = eye(n, m, ..., z, 'like', V)
R = eye(n, m, ..., z, classname)
📥Arguments d'entrée
Paramètre Description
n une variable : matrice n-par-n
m une variable : matrice n-par-m
📄Description

eye retourne une matrice identité.

💡Exemples
eye(3)
eye(3,1,3,'single')
A = single([3 3])
B = eye(2,4,'like', A)
A = eye(0, 4)
