ishandle
Return true if variable var is a handle object.
📝 Syntax 
res = ishandle(var)
📥 Input Arguments
Parameter Description
var a variable
📤 Output Arguments
Parameter Description
res a logical: true or false
📄 Description
ishandle returns a logical 1 if the argument is a handle object and a logical 0 otherwise.
💡 Examples
A = 3; res = ishandle(A)
