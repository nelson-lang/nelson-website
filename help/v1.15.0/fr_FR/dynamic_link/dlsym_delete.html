dlsym_delete
Supprime l'objet dlsym
📝Syntaxe
dlsym_delete(h)
delete(h)
📥Arguments d'entrée
Paramètre Description
h un handle : un objet dlsym.
📄Description

delete(h) libère l'objet dlsym.

N'oubliez pas de nettoyer la variable h ensuite.

💡Exemples
dlsym_used(),delete(dlsym_used())
