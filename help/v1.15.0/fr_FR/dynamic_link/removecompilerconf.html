removecompilerconf
Supprime la configuration du compilateur utilisée (sous Windows)
📝Syntaxe
res = removecompilerconf()
📤Arguments de sortie
Paramètre Description
res a logical
📄Description

removecompilerconf renvoie true si un compilateur avait été configuré avec configuremsvc ou configuremingw.

removecompilerconf renvoie toujours true sur les autres plateformes.
