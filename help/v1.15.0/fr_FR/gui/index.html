Module GUI

Le module GUI fournit des fonctions pour créer et interagir avec les composants d'interface graphique, les boîtes de dialogue et les fenêtres d'application.

  • commandhistory
    Historique des commandes
  • errordlg
    Crée une boîte de dialogue d'erreur.
  • filebrowser
    Explorateur du dossier courant
  • helpdlg
    Crée une boîte de dialogue d'aide.
  • inserthtml
    Insère du HTML dans la console GUI.
  • lookandfeel
    Apparence et comportement (look and feel) de l'application
  • msgbox
    Crée une boîte de dialogue de message.
  • openvar
    Ouvre une variable dans l'éditeur de variables
  • qt_verbose
    Afficher/masquer les messages de débogage Qt.
  • questdlg
    Crée une boîte de dialogue de question.
  • uigetdir
    Ouvre une boîte de dialogue pour sélectionner un répertoire.
  • warndlg
    Crée une boîte de dialogue d'avertissement.
  • workspace
    Explorateur d'espace de travail
