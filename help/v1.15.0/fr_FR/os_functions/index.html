Fonctions du système d'exploitation

Le module Fonctions OS fournit des outils pour interagir avec le système d'exploitation dans Nelson.

Il inclut des fonctions pour interroger les informations système, gérer les variables d'environnement, exécuter des commandes shell, générer des GUID et effectuer des opérations spécifiques à la plateforme.

Ce module permet une intégration fluide des scripts Nelson avec le système d'exploitation sous-jacent sur Windows, macOS et Linux/Unix.

  • cmdsep
    Séparateur de commande pour le système d'exploitation courant.
  • computer
    Informations sur le système.
  • createGUID
    Crée un GUID.
  • getenv
    Obtenir la valeur d'une variable d'environnement.
  • hostname
    obtenir le nom d'hôte de cet ordinateur.
  • ismac
    Vérifie si la version est pour la plateforme macOS.
  • ispc
    Vérifie si la version est pour la plateforme Windows.
  • isunix
    Vérifie si la version est pour une plateforme GNU/Linux ou Unix.
  • searchenv
    Recherche un fichier en utilisant les chemins définis dans une variable d'environnement.
  • setenv
    Définir la valeur d'une variable d'environnement.
  • system
    Exécution de commandes shell.
  • dos
    Exécution de commandes shell.
  • unix
    Exécution de commandes shell.
  • username
    obtenir le nom d'utilisateur courant.
  • winopen
    Ouvrir un fichier dans l'application appropriée (Windows seulement).
  • winqueryreg
    Lire le registre Windows (Windows seulement).
