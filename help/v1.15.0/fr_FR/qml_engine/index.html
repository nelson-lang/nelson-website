Moteur QML

Le module Moteur QML permet aux programmes Nelson d'afficher, manipuler et interagir avec du contenu graphique en utilisant le framework QML de Qt.

Il fournit des fonctions pour gérer les composants QML, accéder aux objets Qt et intégrer la logique JavaScript et QML.

  • QObject_classname
    Renvoie le nom de classe d'une poignée (handle) QObject.
  • QObject_findchildren
    Renvoie tous les enfants de cet objet ayant le nom donné.
  • QObject_get
    Récupère la valeur d'une propriété d'une poignée (handle) QObject.
  • QObject_iswidgettype
    Renvoie true si le QObject est un widget.
  • QObject_iswindowtype
    Renvoie true si le QObject est une fenêtre.
  • QObject_methodsignature
    Renvoie la signature d'une méthode d'une poignée (handle) QObject.
  • QObject_root
    Objet racine QObject.
  • QObject_set
    Définit la valeur d'une propriété d'une poignée (handle) QObject (set).
  • QObject_undefine
    Supprime une propriété dynamique d'une poignée (handle) QObject.
  • QObject_used
    Renvoie la liste des poignées (handles) QObject actuellement utilisées.
  • nelsonObject
    objet nelson appelable depuis QML.
  • qml_addimportpath
    Ajoute un chemin comme répertoire où le moteur QML recherche les modules installés.
  • qml_addpluginpath
    Ajoute un chemin comme répertoire où le moteur QML recherche les plugins natifs.
  • qml_clearcomponentcache
    Vide le cache interne de composants du moteur.
  • qml_collectgarbage
    Exécute le ramasse-miette QML.
  • qml_createqquickview
    Charge un fichier QML et crée une fenêtre.
  • qml_demos
    Démos QML.
  • qml_evaluatefile
    Évalue un fichier JS.
  • qml_evaluatestring
    Évalue une chaîne JS.
  • qml_importpathlist
    Renvoie la liste des répertoires où le moteur recherche les modules installés dans une structure de répertoires basée sur des URL.
  • qml_loadfile
    Charger un fichier QML.
  • qml_loadstring
    Load a QML string.
  • qml_offlinestoragepath
    Obtient la propriété contenant le répertoire pour stocker les données utilisateur hors ligne.
  • qml_pluginpathlist
    Renvoie la liste des répertoires où le moteur recherche les plugins natifs pour les modules importés.
  • qml_setofflinestoragepath
    Définit la propriété contenant le répertoire pour stocker les données utilisateur hors ligne.
  • qt_constant
    Renvoie la valeur d'une constante Qt.
  • qt_version
    Renvoie la version de Qt utilisée.
