stop
Stops an audioplayer object.
📝Syntax
stop(playObj)
📥Input Arguments
Parameter Description
playObj an audioplayer object.
📄Description

stop stops an audioplayer object.

💡Examples
signal = rand(2, 44100) - 0.5;
playObj = audioplayer(signal, 44100, 16)
play(playObj)
sleep(2)
stop(playObj)
delete(playObj)
playObj
