uminus
Unary minus, - operator
📝Syntax
C = uminus(A)
C = -A
📥Input Arguments
Parameter Description
A a variable
📤Output Arguments
Parameter Description
C result of -A
📄Description

C = uminus(A) performs unary minus ie -A.

💡Examples
M = 3;
-M
