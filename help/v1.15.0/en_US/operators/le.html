le
less than or equal, = operator.
📝Syntax
C = le(A, B)
📥Input Arguments
Parameter Description
A a variable
B a variable
📤Output Arguments
Parameter Description
C result of le(A, B)
📄Description

C = le(A, B) returns a logical array with elements set to logical true A is less than or equal to B.

le compares only the real part of numeric arrays.

💡Examples
eye(2,2) &#60;= ones(2, 2)
0 &#60;= i
'Nelson' &#60;= 'Noslen'
'Nelson' &#60;= 'l'
le(0.8 - 0.6 - 0.2, 0)
