System Requirements by platforms.

Linux:

Operating system: Ubuntu 24.04 LTS, Ubuntu 22.04 LTS, Ubuntu 20.04 LTS, Fedora, ArchLinux, NixOs

Processor: Any Intel or AMD x86-64 processor.

RAM: See Operating System Requirements (16 GB recommanded or more).

Storage: 1 GB for all products installation and an SSD/nvme is recommended.

Graphics: A hardware-accelerated GPU with OpenGL support and at least 1GB of memory is recommended.

Windows:

Operating system: Windows 10, 11 or Windows Server 2022.

Processor: Any Intel or AMD x86-64 processor with AVX2 instructions set (CPU released from 2015 onwards).

RAM: See Operating System Requirements (16 GB recommanded or more).

Storage: 1 GB for all products installation and an SSD/nvme is recommended.

Graphics: A hardware-accelerated GPU with OpenGL support and at least 1GB of memory is recommended.

MacOs:

Operating system: macOS Tahoe, macOS Sequoia, macOS Sonoma, macOS Ventura.

Processor: Any Intel processor support by Apple and any M-series chip.

Storage: 1 GB for all products installation and an SSD/nvme is recommended.

Graphics: Any Mac capable of running macOS Ventura has a GPU that can run Nelson.
