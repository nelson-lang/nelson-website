webwrite
Write data to RESTful web service
üìùSyntax
response = webwrite(url, data)

üì•Input Arguments
Parameter Description

üì§Output Arguments
Parameter Description
response a variable: Response from a web service.

webwrite write data to RESTful web service.

Connect to your NetAtmo Weather station (oAuth2 connection)
USER_NAME = 'your username'; PASSWORD = 'your password'; CLIENT_ID = 'your client id'; CLIENT_SECRET = 'your client secret'; DEVICE_ID = 'your device id'; url = 'https://api.netatmo.com/oauth2/token'; args = {'grant_type', 'password', 'username', USER_NAME, 'password', PASSWORD, 'client_id', CLIENT_ID, 'client_secret', CLIENT_SECRET}; response = webwrite(url, args{:}); r = webread('https://api.netatmo.com/api/getstationsdata', 'access_token', response.access_token, 'device_id', DEVICE_ID); disp('Devices') disp(r.body.devices) disp('Location Info') disp(r.body.devices.place) disp('Last values') disp(r.body.devices.dashboard_data)
